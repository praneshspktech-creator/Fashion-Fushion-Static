Fashion Fusion - React + Tailwind project skeleton
=================================================

What is included:
- Vite + React project skeleton (JSX)
- TailwindCSS configuration files
- Basic components: Navbar, Footer, CategoryCard, ValuePropCard, Testimonial
- Home page implemented to match the provided requirements document
- Placeholder assets folder for category images

How to run locally:
1. cd fashion-fusion
2. npm install
3. npm run dev

Notes:
- This is a starter skeleton. Add images in src/assets and refine styles / responsive behavior as required.
- The package.json uses conservative dependency ranges; adjust versions to your preference.

Source requirements: The project skeleton was created from the uploaded requirements document (Fashion Fusion.docx).
